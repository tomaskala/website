- Anathem
  - Thinking = the brain processing sensory inputs, converting them to a common 
    representation and using it to make inference in order to perform an action
  - Consciousness
    - Thinking ~ wave function collapse
    - We're toying with several ideas at once, considering them from several 
      angles when suddenly, the solution falls into place and becomes crystal 
      clear
    - Many-worlds interpretation: there are multiple copies of the mind, each 
      with a model of a slightly different universe, using it to make 
      inference. How would we store a model of the entire universe inside the 
      mind?
      - Assume instead that there are many slightly different universes, each 
        with a copy of the mind. At that point, we don't even need the model 
        anymore, it's there in front of our eyes.
      - Analogy of the wave function collapse is that the universe leading to 
        the solution becomes the main narrative that we live in. The other 
        universes do not necessarily vanish, it's just that we happen to live 
        in that particular universe (we have to pick one, after all).
    - This doesn't mean that the brain is somehow detached from the rest of the 
      universe and being copied across multiple parallel worlds with everything 
      else remaining intacts. The exact opposite, actually: the entire universe 
      is being copied, it's just that the brain has developed a way to utilize 
      this fact.
- The Emperor's New Mind
  - Can we simulate the human mind using a computational process? Probably not.
  - Any computer program, regardless of how complex it is, can be reduced to a 
    Turing machine.
    - Any Turing machine can be described with a set of mathematical formulae.
    - Simulating a conscious being using a computer program is therefore 
      equivalent to sitting down and producing a set of specifications using 
      mathematical notation. Claiming that a stack of papers with some math on 
      them contains the description of the entire human mind sounds absolutely 
      ridiculous. By following them, we could perform an action equivalent to 
      thinking, but this reduces to an instance of the Chinese room.
